# 파이썬 루트 잡아주기

가상환경을 나누어 쓰고 있으니, 따로 잡아줘야 한다.

preferences > Project:... > Project Interpreter > 설정 버튼 > add local > ~ -.pyenv-versions-pinkfongstore-bin-python을 눌러준다.

우리가 설치한 라이브러리들을 보려면, Exteral Libraries > site-packages 를 보면 나온다.
